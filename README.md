## Hướng dẫn chạy
1. Tải source code về máy.
2. Mở file `index.html` bằng trình duyệt (Chrome, Edge, Firefox...).
3. Nhấn phím `F12` (hoặc chuột phải -> Inspect/Kiểm tra).
4. Chọn tab **Console** để xem kết quả chi tiết của 10 câu hỏi.
